Capítulo 1

HTML: A Linguagem da Estrutura

O que é? HTML (HyperText Markup Language) é o esqueleto da web. Ele não programa, ele descreve. Com "tags" (etiquetas), você marca seu conteúdo, dizendo ao navegador: "Isto é um título", "Isto é uma imagem", "Isto é um link".

Tags Essenciais do Dia a Dia:

  • <h1> a <h6>: Títulos e subtítulos.
  • <p>: Parágrafos de texto.
  • <a href="https://...">: Links clicáveis.
  • <img src="imagem.jpg" alt="descrição">: Imagens (o `alt` é obrigatório para acessibilidade).
  • <div>: Uma "caixa" genérica para agrupar elementos.
Capítulo 2

CSS: A Linguagem do Estilo

O que é? CSS (Cascading Style Sheets) é a linguagem de design da web. Se o HTML é o esqueleto, o CSS é a roupa. Ele seleciona elementos HTML e aplica regras de estilo a eles.

O Box Model: O Conceito MAIS Importante do CSS

Todo elemento na tela é uma caixa. De dentro para fora, a caixa é composta por:

  1. Conteúdo: O texto, a imagem, etc.
  2. Padding: Um espaçamento interno.
  3. Border: A borda da caixa.
  4. Margin: Um espaçamento externo, entre as caixas.
Capítulo 3

JavaScript: A Linguagem da Interatividade

O que é? JavaScript (JS) é uma linguagem de programação que roda no navegador. Ele é o "cérebro" da página. O JS pode ouvir eventos (cliques, teclado) e alterar o HTML e o CSS em tempo real.
