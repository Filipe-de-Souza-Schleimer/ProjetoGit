O Universo do Desenvolvimento Web

Uma introdução aos três pilares da web: HTML, CSS e JavaScript.

HTML: A Estrutura da Página

HTML (HyperText Markup Language) é a linguagem de marcação que usamos para estruturar o conteúdo de uma página web. Pense nele como o esqueleto de um corpo.

<h1>Este é um título</h1>
<p>Este é um parágrafo.</p>
<button>Isto é um botão</button>

Cada elemento, como títulos, parágrafos ou imagens, é definido por "tags" (etiquetas).

CSS: O Estilo e a Aparência

CSS (Cascading Style Sheets) é a linguagem que usamos para estilizar a nossa página. Se o HTML é o esqueleto, o CSS é a roupa, a pele, o cabelo e a maquiagem.

h1 {
    color: #2c3e50; /* Cor do texto */
}

button {
    background-color: #3498db; /* Cor de fundo */
    color: white; /* Cor do texto */
    border-radius: 8px; /* Bordas arredondadas */
}

Com CSS, definimos cores, fontes, espaçamentos, layout e muito mais.

JavaScript: A Interatividade

JavaScript (JS) é a linguagem de programação que dá vida à página. É o cérebro e os músculos, permitindo que a página responda às ações do usuário.

document.getElementById('meuBotao').onclick = function() {
    alert('Você clicou no botão!');
};

Ele manipula o HTML e o CSS para criar interações, validar formulários, carregar dados e muito mais.
