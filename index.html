Dia 1: HTML - A Estrutura Semântica

HTML (HyperText Markup Language) é a espinha dorsal da web. A versão 5 (HTML5) introduziu tags semânticas que dão significado ao conteúdo, auxiliando na acessibilidade e SEO.

  • <header>: Para o topo do site ou de uma seção.
  • <nav>: Para menus de navegação principais.
  • <main>: Delimita o conteúdo principal e único da página.
  • <article>: Para um bloco de conteúdo autocontido (um post de blog, uma notícia).
  • <aside>: Para conteúdo lateral, como uma barra de informações.
  • <footer>: Para o rodapé do site ou de uma seção.

Usar a tag certa para o trabalho certo é o primeiro passo para se tornar um desenvolvedor profissional.

Dia 2: CSS - Estilização e Layouts Responsivos

CSS (Cascading Style Sheets) evoluiu de simples regras de cor para sistemas de layout complexos. Hoje, dominamos a aparência de um site com ferramentas como Flexbox e Grid Layout.

O Design Responsivo é crucial. Usando Media Queries, podemos adaptar o layout para que ele seja perfeito em qualquer dispositivo, do celular ao desktop.

/* Exemplo de Media Query */
@media (max-width: 768px) {
  .layout-grid {
    grid-template-columns: 1fr; /* Coluna única em telas pequenas */
  }
}

Dia 3: JavaScript - Dominando o DOM

JavaScript (JS) nos permite manipular o Document Object Model (DOM) - a representação em árvore do nosso HTML. Em vez de modificar o HTML diretamente, usamos JS para reagir a eventos.

A forma moderna de fazer isso é adicionando ou removendo classes CSS com element.classList.toggle('minha-classe'). Isso mantém a lógica (JS) separada da estilização (CSS).

const botao = document.querySelector('#meuBotao');
const elemento = document.querySelector('.alvo');

botao.addEventListener('click', () => {
  elemento.classList.toggle('ativo');
});
